### Projects

**Quadruped Navigation — Unitree Go2**

* Implemented goal-based navigation, enabling the quadruped to autonomously move toward a specified target pose.
* Integrated an Xbox controller interface for real-time teleoperation and manual override.

**Bipedal Locomotion — Agility Robotics Digit**

* Contributed to the initial system setup for bipedal traversability estimation using Model Predictive Control (MPC).
* Implemented obstacle perception modules for terrain awareness.
* Designed and generated challenging terrains in MuJoCo to evaluate locomotion stability and robustness.

**Current & Future Work — Unitree G1 (Isaac Lab)**

* **G1 Locomotion and Interaction Training** :

  Developing a learning-based locomotion framework in Isaac Lab, with the primary objective of achieving stable and efficient walking

* **Traversability Neural Network Training** :

  Training a neural network to estimate terrain traversability for the G1, supporting informed footstep planning and motion decisions.

* **Terrain Inpainting Network** :

  Implementing a neural inpainting model to reconstruct terrain representation for alternative perception and planning.
